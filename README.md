## Projetos

### 🤖 [Agente de Mediações — Mercado Livre](https://github.com/kauasilvadsantos/agente-mediacoes-z4auto)

Agente de IA (Google Gemini) que gera defesas estratégicas para disputas no Mercado Livre, respeitando o limite de **320 caracteres** da plataforma. Histórico de casos persistido em Firebase Firestore, 3 estratégias por disputa, fallback offline e biblioteca de templates para casos recorrentes.

### 📦 [Monitor de Ordens de Compra — ERP Tiny](https://github.com/kauasilvadsantos/monitor-ordens-compra-tiny)

App desktop Python que monitora ordens de compra no ERP Tiny e alerta sobre entregas **atrasadas** (🔴) ou **próximas do vencimento** (🟡). Autenticação via OAuth2, renovação automática de token, paginação da API e modo `--auto` para agendamento diário. Build para `.exe` standalone com PyInstaller.
